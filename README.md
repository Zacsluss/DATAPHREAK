### **Why You'll Love the Architecture**

```bash
# Clone and use immediately - no build process
git clone https://github.com/Zacsluss/DATAPHREAK.git
cd DATAPHREAK
# Open DATAPHREAK.html in browser - that's it!
```

**Key Design Decisions**:
- **Single HTML file**: Eliminates dependency hell, version conflicts, build pipeline complexity
- **Embedded libraries**: SheetJS baked in = true offline capability, no CDN single point of failure
- **Pure JavaScript**: No frameworks = fast load times, easy debugging, transparent execution
- **CSS Variables**: Theme system uses native CSS custom properties = instant theme switching
- **localStorage only for prefs**: ZERO data persistence = privacy by architecture, not promise

### **Code Highlights**

```javascript
// Fuzzy matching with smart blocking for O(n log n) performance
function fuzzyMatch(str1, str2, threshold = 0.8) {
  const distance = levenshteinDistance(str1, str2);
  const maxLen = Math.max(str1.length, str2.length);
  return (1 - distance / maxLen) >= threshold;
}

// Quality scoring algorithm (Completeness + Consistency + Validity)
const qualityScore = (completeness * 0.4) + (consistency * 0.3) + (validity * 0.3);
const grade = qualityScore >= 0.9 ? 'A' : /* grading logic */ 'F';
```
